#  Superstore Data Analysis Project

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Superstore dataset using Python. The goal is to analyze sales performance, profit trends, customer behavior, and regional performance to extract meaningful business insights.

##  Dataset
- Source: Kaggle Superstore Dataset  
- File: Sample - Superstore.csv  
- Contains information about orders, sales, profit, customers, products, and regions.
## Objectives
- Load and explore the dataset  
- Clean and preprocess data  
- Perform feature engineering  
- Analyze sales and profit trends  
- Use groupby and pivot tables for insights  
- Visualize key business patterns  
- Extract actionable insights  
## Tools Used
- Python 🐍  
- Pandas 📊  
- NumPy 🔢  
- Matplotlib 📈  
- Jupyter Notebook 📓  
##  Analysis Performed
### 1. Data Loading & Exploration
- Loaded dataset using Pandas  
- Explored structure using `.info()` and `.describe()`  
- Checked data types and missing values  
### 2. Data Cleaning
- Handled missing values  
- Converted date columns to datetime format  
- Ensured data consistency  

### 3. Feature Engineering
- Extracted Year and Month from Order Date  
- Created new derived features for analysis  

### 4. GroupBy & Aggregation
- Total sales by category  
- Total profit by region  
- Top 5 customers by sales  
- Monthly sales trends  

### 5. Pivot Tables
- Sales by category and region  
- Sales by segment over time  
- Profit by sub-category  


### 6. Data Visualization
- Line chart: Sales trends over time  
- Bar chart: Sales by category  
- Bar chart: Profit by region  
- Pie chart: Segment distribution  


##  Key Insights
- Technology and Office Supplies perform strongly in sales and profit  
- Some regions show high sales but low profitability  
- Sales vary significantly across months, showing seasonal trends  
- A small number of customers contribute a large portion of total sales  


##  Conclusion
This project provides valuable insights into business performance, helping understand sales trends, customer behavior, and regional performance. It can support better decision-making in marketing and operations.
